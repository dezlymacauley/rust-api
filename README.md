# Rust API
_______________________________________________________________________________

## Starting the server

Open a separate terminal and run this command

```bash
cargo dev
```
_______________________________________________________________________________

## Testing the server

Open a separate terminal and navigate the `api-testing` directory

You can send a request to the root of the API like this:

```bash
hurl root_get.hurl
```
